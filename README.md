Predictive Maintenance AI System
 Overview
This project is a Predictive Maintenance System built using Machine Learning and Python.
It analyzes real-time sensor data (temperature, vibration, pressure) to detect anomalies and predict potential machine failures.

The system helps industries reduce downtime, prevent breakdowns, and optimize maintenance scheduling.

 Problem Statement

Traditional maintenance methods:

Reactive (after failure)

Scheduled (fixed intervals)

Both increase cost and downtime.

This project introduces:

. Data-driven predictive maintenance using anomaly detection.
